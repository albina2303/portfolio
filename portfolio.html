Currency converter

This is a web-based application to help you convert currencies quickly and easily. You will be able to select the source and target currencies, enter the amount to convert and my application will instantly show you the equivalent amount in the selected currency. I implemented it using HTML, CSS and JavaScript. The app can use the API to retrieve the current exchange rates or let you update the rates manually. I am sure that Currency Converter will be a handy tool for anyone who needs fast and accurate currency conversion!

Weather app

If you are interested in knowing the current weather and the forecast for the next few days, this app is for you. You will be able to enter a city name or use geolocation to get weather data for your area. My app will show you the temperature, weather conditions, humidity and other indicators to help you prepare for the day. I created it using HTML, CSS and JavaScript and integrated an API to get accurate and up-to-date weather information. I hope that the Weather App will become your trusted companion in everyday life!

Calorie calculator

If you want to monitor your diet and take care of your health, this app is perfect for you. I designed the Calorie Calculator using HTML, CSS and JavaScript to make it intuitive and easy to use. I am confident that this pet project will become your reliable assistant on your way to a healthy lifestyle!
